@@iterador

El método @@iterator es parte del protocolo iterable, que define cómo iterar sincrónicamente sobre una secuencia de valores.

El valor inicial de la propiedad "@@iterator" es el mismo objeto de función que el valor inicial de la propiedad "values()".

valores() El método de "values()" devuelve un nuevo objeto iterador de matriz que contiene los valores para cada índice en la matriz.

Un detalle importante es que los números no son iterables. Bueno, en realidad los números, que son tratados como números, pues en JavaScript si encierras números entre comillas JavaScript los tratará como texto. La realidad es que JavaScript lo trata todo como texto en un principio o por lo menos todos los valores extraídos de campos input. Esto se debe a que JavaScript es un lenguaje no tipado, o sea que no maneja tipos de datos en un principio. Hay métodos para ayudarnos a encontrar el tipo de dato al que pertenece un dato de entrada, por el usuario o recuperado de una base de datos. Por ejemplo, 123 no es igual que "123" aunque en ocasiones JavaScript al compararlos, puede decir que son iguales, de ese tema hablaremos después, ahora iremos al ejemplo de iterador en JavaScript.
