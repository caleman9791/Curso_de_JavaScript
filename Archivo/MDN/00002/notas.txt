Instalación de software básico

Cuando se trata de herramientas para construir un sitio web, hay mucho que elegir. Si está comenzando, puede estar confundido por la matriz de editores de código, marcos y herramientas de prueba que hay. Al instalar el software básico, le mostraremos paso a paso cómo instalar el software que necesita para comenzar un desarrollo web básico.

¿Cómo se veía su sitio web?

Antes de comenzar a escribir el código para su sitio web, debe planificarlo primero. ¿Qué información estás mostrando? ¿Qué fuentes y colores estás usando? ¿Cómo se veía su sitio web? Describimos un método simple que puede seguir para planificar el contenido y el diseño de su sitio.
Tratar con archivos

Un sitio web consta de muchos archivos: contenido de texto, código, hojas de estilo, contenido de medios, etc. Cuando está construyendo un sitio web, debe reunir estos archivos en una estructura sensible y asegurarse de que puedan hablar entre sí. Tratar con los archivos explica cómo configurar una estructura de archivos sensible para su sitio web y en qué problemas debe conocer.


Conceptos básicos de HTML

El lenguaje de marcado de hipertexto (HTML) es el código que utiliza para estructurar su contenido web y darle un significado y un propósito. Por ejemplo, ¿es mi contenido un conjunto de párrafos o una lista de puntos de bala? ¿Tengo imágenes insertadas en mi página? ¿Tengo una tabla de datos? Sin abrumadoras, los conceptos básicos de HTML proporcionan suficiente información para que esté familiarizado con HTML.


Conceptos básicos de CSS

Las hojas de estilos de cascada (CSS) son el código que utiliza para diseñar su sitio web. Por ejemplo, ¿quieres que el texto sea negro o rojo? ¿Dónde debe dibujarse el contenido en la pantalla? ¿Qué imágenes y colores de fondo deben usarse para decorar su sitio web? Los conceptos básicos de CSS lo llevan a través de lo que necesita para empezar.


Conceptos básicos de JavaScript

JavaScript es el lenguaje de programación que usa para agregar funciones interactivas a su sitio web. Algunos ejemplos podrían ser juegos, las cosas que ocurren cuando se presionan los botones o se ingresan datos en formas, efectos de estilo dinámico, animación y mucho más. JavaScript es básico para brindarle una idea de lo que es posible con este emocionante idioma, y cómo empezar.


Publicando su sitio web

Una vez que haya terminado de escribir el código y organizar los archivos que conforman su sitio web, debe ponerlo todo en línea, para que la gente pueda encontrarla. La publicación de su código de muestra describe cómo obtener su código de muestra simple en línea con un esfuerzo mínimo.


Cómo funciona la web

Cuando accede a su sitio web favorito, muchas cosas complicadas ocurren en el fondo que no puede saber. Cómo funciona la web describe lo que sucede cuando ve una página web en su computadora.

Como ya saben la creación de aplicaciones web es muy semejante a la construcción de un edificio. Tal vez  puedas generar más ingresos de esta forma, pero tal vez no puedas cubrir las necesidades personalizadas que necesita una empresa.
