JavaScript es un lenguaje de programación usado en el desarrollo de aplicaciones web. Este tiene una popularidad increíble dentro del entorno de desarrollo de interfaces gráficas e interactividad, lo cual son interpretados por un navegador web. JavaScript es interpretado por la mayoría de los navegadores web por no decir todos. La compatibilidad con todos los navegadores no siempre es la esperada, pues los navegadores deben implementar los estándares antes de que puedan ser interpretados. Cada vez es más uniforme esta compatibilidad. Sin embargo, es el lenguaje número 1 en el desarrollo de interfaces de usuario pensadas para la web.

Fue creado en 1995 por Brendan Eich mientras trabajaba en Netscape Communications Corporation. JavaScript ha evolucionado mucho desde su creación. Hoy día, es un lenguaje muy poderoso y completo. Posee muchas herramientas y bibliotecas disponibles, lo cual amplifica su potencial y facilita su implementación. Por otro lado, cuenta con una amplia comunidad de desarrolladores que trabajan en mejoras y en la expansión del lenguaje y su ecosistema.

¿Por qué estudiar JavaScript?

JavaScript es uno de los 3 idiomas que todos los desarrolladores web deben aprender:

  1. HTML para definir el contenido de las páginas web
  2. CSS para especificar el diseño de las páginas web
  3. JavaScript para programar el comportamiento de las páginas web
